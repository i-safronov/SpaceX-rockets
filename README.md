# SpaceX-rockets
The application shows information about SpaceX space rockets and a list of their launches

## Completed the test task of the company «Контур»

### Technology Stack:
- Kotlin
- Kotlin Coroutines 
- Retrofit
- Room
- Jetpack Navigation
- Koin
